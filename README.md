# Footprint Logger Platform

A full-stack web application for tracking daily carbon emissions, visualizing data, and receiving personalized reduction tips.

## Features

- User registration & JWT authentication
- Log activities (transport, food, energy) with predefined CO₂ values
- Real-time running total and pie chart visualization
- Filter activities by category
- Dashboard with weekly summaries, streaks, and community average comparison
- Personalized tips based on highest emission category

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript, Chart.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Authentication**: JWT
